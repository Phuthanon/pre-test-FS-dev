docker image จะเป็นตัวต้นแบบในการสร้าง container 
ที่ประกอบไปด้วยแอพลิเคชั่นต่างๆ ซึ่งจะถูกเรียกใช้งานต่อเมื่อ 
Docker Container ทำงาน ตัวอย่างเช่น Docker image 
เป็นรูปแบบของบ้าน ส่วน Docker container คือบ้านหลายๆหลังที่สร้างขึ้นมา
ตามแบบของ Docker image ซึ่งแต่ละบ้านจะทำงานโดยไม่มีความเกี่ยวข้องกัน
